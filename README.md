# card-reports

Rendered HTML reports from the (private) card-arbitrage pipeline, published
via GitHub Pages so they can be read from a bookmark — no GitHub account
needed.

- **Latest weekly report:** https://averycpawelek.github.io/card-reports/
- **Criteria tuning brief:** https://averycpawelek.github.io/card-reports/criteria-brief.html
- **Past weeks:** `archive/YYYY-MM-DD.html` under the same URL

This repo holds only rendered output. It is written to automatically by the
weekly GitHub Action in the private pipeline repo (via a deploy key); nothing
here is edited by hand. All pages carry `noindex, nofollow`.
